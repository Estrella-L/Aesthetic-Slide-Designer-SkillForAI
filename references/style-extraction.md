# 风格提取参考清单

从图片或网页提取审美风格时，逐项过一遍，整理成一份 Style Spec。不确定的维度做合理推断并标注"推断"。维度越全，复刻越像。

## 1. 配色 (Color)

- 主色 (primary)：占比最大、定义整体气质的颜色。
- 辅助色 (secondary)：与主色搭配的次要色。
- 强调色 (accent)：用于按钮/高亮/关键数字的点缀色。
- 中性色：背景色、卡片色、分隔线色。
- 文字色：主文字、次要文字、反白文字。
- 从图片取色时给出近似 HEX；从网页取色时直接读 CSS 里的色值。
- 配色关系：单色 / 邻近色 / 互补色 / 三角配色 / 高饱和 / 低饱和(灰调) / 莫兰迪 / 霓虹 / 大地色 / 粉彩。
- 明暗模式：亮色 / 暗色 / 高对比 / 低对比柔和。
- 配色温度：暖调 / 冷调 / 中性。

## 2. 字体 (Typography)

- 标题字族与正文字族（衬线 / 无衬线 / 等宽 / 手写 / 展示体 / 像素 / 哥特 / 圆体）。
- 中英文混排：中文字族（黑体 / 宋体 / 楷体 / 圆体）与西文搭配。
- 字重层级（如 标题 700、副标题 500、正文 400）。
- 字号比例（type scale 倍率，如 1.2 / 1.25 / 1.333 / 1.618）。
- 行高、字间距、段间距；是否全大写、是否字偶距夸张。
- 排版气质：紧凑工整 / 大开大合 / 杂志编辑感 / 极简留白。
- 网页输入：直接从 `font-family`、`@font-face`、Google Fonts 链接读取。

## 3. 布局与间距 (Layout & Spacing)

- 内容密度：紧凑(信息密) / 疏朗(大留白) / 极简(单点聚焦)。
- 对齐：居中 / 左对齐 / 右对齐 / 网格化 / 非对称 / 自由错落。
- 栅格系统：列数、栏宽、装订线；是否打破网格。
- 间距节奏：是否有统一 spacing 单位（如 4px / 8px 基准）。
- 内容区宽度、页边距、安全区。
- 构图：黄金分割 / 三分法 / 对称 / 满版出血 / 中心放射。

## 4. 情绪基调 (Mood)

用 2-4 个形容词概括。常见基调坐标：
- 冷静 ↔ 热烈
- 严肃专业 ↔ 活泼趣味
- 极简克制 ↔ 繁复华丽
- 高级奢华 ↔ 平价亲民
- 理性科技 ↔ 感性人文
- 现代未来 ↔ 复古怀旧

## 5. 细节语言 (Detail Language)

- 圆角：尖角 / 小圆角 / 大圆角 / 胶囊形 / 混合。
- 阴影：无 / 微阴影 / 强投影 / 内阴影 / 彩色光晕。
- 边框与分隔线：有无、粗细、虚实、颜色。
- 装饰元素：渐变 / 网格线 / 噪点 / 几何形 / 流体形 / 手绘涂鸦 / 贴纸 / 引号大字。
- 质感：扁平 / 拟物 / 玻璃拟态(毛玻璃) / 新拟物(Neumorphism) / 纸质 / 金属 / 渐变光感。

## 6. 图像与插画 (Imagery & Illustration)

- 图片处理：满版出血 / 留边卡片 / 圆形裁剪 / 拼贴 / 双色调(duotone) / 滤镜色调 / 黑白。
- 插画风格：扁平矢量 / 线描 / 3D 渲染 / 等距(isometric) / 手绘水彩 / 像素 / 抽象几何。
- 图标风格：线性 / 面性 / 双色 / 拟物 / emoji 化。
- 图文关系：图主文辅 / 文主图辅 / 图文穿插。

## 7. 动效与交互 (Motion) —— 若参考为网页

- 入场动效：淡入 / 上滑 / 缩放 / 逐字 / 视差滚动。
- 微交互：悬停反馈、按钮态、光标特效。
- 节奏：克制顺滑 / 活泼弹性 / 夸张戏剧。
- 转场风格：硬切 / 渐隐 / 推拉 / 翻页。

## 8. 数据可视化风格 (Data Viz) —— 若含图表

- 图表类型偏好、配色映射、是否极简去网格。
- 数字呈现：超大号强调数字 / 配单位标签 / 进度环。

## 9. 文化与时代参照 (Reference)

- 时代感：80s 复古 / 90s Y2K / 千禧粉 / 当代极简。
- 文化语境：日式侘寂 / 北欧极简 / 包豪斯 / 孟菲斯 / 国潮 / 赛博朋克 / 编辑杂志。
- 品牌联想：是否像某类知名产品/媒体的视觉语言（描述气质即可，不抄袭）。

## 风格原型库 (Style Archetypes)

识别参考接近哪种原型，可快速锁定方向（可组合）：

- **极简留白 (Minimal)**：大量空白、细字体、克制配色、几乎无装饰。
- **科技未来 (Tech / Futuristic)**：深色底、霓虹强调色、等宽字体、网格线、光晕。
- **编辑杂志 (Editorial)**：衬线大标题、非对称网格、强烈字号对比、黑白照片。
- **玻璃拟态 (Glassmorphism)**：毛玻璃半透明卡片、彩色渐变背景、柔光。
- **新粗野 (Neo-Brutalism)**：高饱和撞色、粗黑边框、硬阴影、无圆角、大字。
- **孟菲斯 (Memphis)**：撞色、几何碎片、波点条纹、活泼随性。
- **Y2K / 千禧**：金属质感、渐变、星芒、果冻按钮、像素元素。
- **日式侘寂 / 性冷淡**：低饱和大地色、大留白、细线、安静。
- **奢华高级 (Luxury)**：黑金 / 深色 + 金属色、衬线、宽字距、克制。
- **手作温暖 (Hand-crafted)**：手写体、纸质纹理、暖色、涂鸦插画。
- **企业专业 (Corporate)**：蓝色系、无衬线、规整网格、卡片化。

## 输出格式示例 (Style Spec)

```
原型: 编辑杂志风 + 极简留白
配色: 主色 #1A1A2E / 强调 #E94560 / 背景 #F5F5F5 / 文字 #16213E (低饱和, 暖调)
字体: 标题 "Playfair Display" 700 衬线, 正文 "Inter" 400, type scale 1.333
布局: 大留白, 左对齐非对称, 8px 基准, 内容宽 880px
基调: 高级 / 克制 / 理性
细节: 无圆角, 1px 细分隔线 #ddd, 无阴影, 扁平质感
图像: 黑白满版照片 + 留边图文
动效: 克制淡入 + 轻视差(网页参考时)
```
